## Zen Browser extensions

Install these manually from official sources. Do not commit Zen's live
`extensions.json` or extension storage files; they can expose profile metadata,
site configuration, and account state.

- Tridactyl
- uBlock Origin
- 1Password
- Firefox Multi-Account Containers
- Zotero Connector
- Dark Reader
- FoxyProxy
- Proton Pass
- Tampermonkey
- SponsorBlock
- Video Speed Controller

Prefer Tridactyl as the primary keyboard layer. Avoid running Vimium and
Tridactyl together long-term unless one of them is disabled for the same pages,
because both extensions compete for normal-mode keys.
